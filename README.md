# Conversion Funnel Analysis

## Overview
This project analyzes user behavior across different stages of a conversion funnel to understand where users drop off before completing a purchase.

---

## 🎯 Objective
- Track user progression through key funnel stages  
- Identify points of drop-off  
- Suggest improvements to increase conversion  

---

## 🛠️ Tools Used
- SQL (MySQL)
- Python (Pandas, Matplotlib)
- Power BI  

---

## 🔄 Workflow

### 1. Data Preparation (SQL)
Created a dataset of user events such as visit, product view, add to cart, and purchase.

### 2. Data Transformation
Converted event-level data into a user-level funnel using SQL aggregation.

### 3. Analysis
Calculated:
- Total users at each stage  
- Conversion rate  
- Drop-offs between stages  

### 4. Visualization (Python)
Visualized user flow across funnel stages to better understand drop-off patterns.

---

## 📊 Key Insight
The largest drop-off occurs between **Product View → Add to Cart**, indicating a gap between user interest and action.

---

## 💡 Recommendations to Improve User Experience
- Improve product page experience  
- Introduce pricing incentives or offers  
- Highlight reviews and ratings  
- Simplify the add-to-cart process
